## Description   
* _docs_   
This directory is documents directory if the project.   
* _rsc(resource)_   
This directory is dictionary or meta data.   
* _src(source)_   
This directory is main source code.   
* _package_   
This directory is the package of the main code.   
* _tests_   
This is test directory.
